# Cloud Native Monitoring Lab 🚀

A complete DevOps and Cloud-Native monitoring practice project built using Kubernetes, Docker, GitHub Actions, Prometheus, Grafana, Loki, Helm, and Ingress.

---

## 📌 Project Overview

This project demonstrates a complete cloud-native DevOps workflow including:

- Containerization using Docker
- Image hosting using Docker Hub
- Kubernetes deployment
- Helm package management
- Monitoring with Prometheus
- Visualization with Grafana
- Logging with Loki
- Ingress configuration
- CI/CD using GitHub Actions

The project was created as part of hands-on DevOps and observability practice.

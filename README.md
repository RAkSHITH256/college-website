# 🎓 College Website DevOps Project

A complete end-to-end DevOps implementation for a College Website application using modern cloud-native tools and workflows.

---

# 🚀 Project Overview

This project demonstrates how real companies build, containerize, deploy, monitor, and automate applications using DevOps practices.

The application is deployed on Kubernetes with automated CI/CD pipelines, monitoring, and observability tools.

---

# 🛠️ Technologies Used

## Development & Version Control
- Git
- GitHub

## Containerization
- Docker
- Docker Hub

## Container Orchestration
- Kubernetes (Kind Cluster)

## CI/CD
- GitHub Actions
- Self-Hosted Runner

## Monitoring & Observability
- Prometheus
- Grafana

## Deployment & Packaging
- Helm

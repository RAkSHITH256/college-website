# Cloud Native Monitoring Lab 🚀

A complete cloud-native DevOps and observability practice project built using Kubernetes, Docker, GitHub Actions, Helm, Prometheus, Grafana, Loki, and Ingress.

---

# 📌 Project Overview

This project demonstrates a complete DevOps workflow including:

* Source code management using Git & GitHub
* Containerization using Docker
* Image hosting using Docker Hub
* Kubernetes deployment and orchestration
* Helm-based package management
* Monitoring using Prometheus
* Visualization using Grafana
* Centralized logging using Loki
* CI/CD automation using GitHub Actions
* Ingress-based routing

This project was created for hands-on practice in DevOps, cloud-native monitoring, Kubernetes operations, and observability engineering.
